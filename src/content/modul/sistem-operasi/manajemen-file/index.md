## Menyimpan dan Mengatur Data

Setiap hari kita membuat, mengunduh, dan menghapus data di komputer. Agar miliaran data ini tidak berantakan, Sistem Operasi menggunakan sistem manajemen file yang menyerupai **lemari arsip di dunia nyata**.

### Direktori (Folder)
Direktori atau Folder adalah tempat kita mengelompokkan file. Di sistem operasi seperti Windows atau Linux, folder-folder ini tersusun seperti struktur pohon terbalik (Root Directory).

- Di **Windows**, strukturnya berawal dari Partisi Drive (Misal: `C:\` atau `D:\`).
- Di **macOS & Linux**, strukturnya berawal dari titik akar tertinggi yang disebut _Root_ (`/`).

### Memahami Ekstensi File
Komputer membedakan jenis dokumen dari **Ekstensi File** (beberapa huruf yang ada di belakang titik pada nama file). Ekstensi inilah yang memberi tahu OS aplikasi apa yang harus digunakan saat file tersebut diklik dua kali.

Beberapa ekstensi populer:
- **Teks/Dokumen:** `.txt`, `.pdf`, `.docx`, `.xlsx`
- **Gambar:** `.jpg`, `.png`, `.gif`, `.svg`
- **Audio/Video:** `.mp3`, `.wav`, `.mp4`, `.mkv`
- **Program yang bisa dijalankan:** `.exe` (Windows), `.app` (Mac), `.sh` (Linux)

> [!TIP]
> Secara default, Windows dan macOS sering menyembunyikan ekstensi file agar terlihat lebih bersih. Namun, menyalakan fitur "Show File Extension" sangat dianjurkan agar kamu tidak tertipu (misalnya ada virus menyamar menjadi gambar dengan nama file `foto.jpg.exe`).
