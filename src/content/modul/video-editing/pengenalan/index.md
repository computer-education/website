## Apa itu Video Editing?

Jika memotret adalah mengumpulkan bahan baku, maka **Video Editing** adalah proses meracik bahan baku tersebut menjadi sebuah hidangan yang lezat. Proses ini sering disebut sebagai tahap _Post-Production_ (Pasca-Produksi).

Apapun software yang kamu gunakan—mulai dari Adobe Premiere Pro, DaVinci Resolve, Final Cut, hingga CapCut di PC/HP—semuanya memiliki elemen antarmuka utama yang sama.

### 3 Elemen Utama Workspace

1. **Media Pool / Project Panel:** 
   Tempat kamu mengumpulkan semua "bahan baku". Mulai dari klip video mentah (_footage_), file musik (_audio_), efek suara (SFX), hingga gambar/logo pendukung.
   
2. **Timeline:**
   Ini adalah "meja operasi"-mu. Timeline adalah area memanjang di bagian bawah layar tempat kamu menyusun video dari kiri ke kanan berdasarkan urutan waktu. Di sinilah proses memotong, menggeser, dan menggabungkan klip dilakukan.
   
3. **Preview / Program Monitor:**
   Layar pemutar tempat kamu melihat hasil sementara dari apa yang sedang kamu susun di Timeline.

> [!TIP]
> **Manajemen File Sangat Penting!**
> Sebelum mulai mengedit, biasakan untuk membuat folder yang rapi di komputermu (misal: Folder `Video Mentah`, `Audio`, `Aset Logo`). Jangan biarkan file berserakan di folder _Downloads_, karena jika file tersebut terhapus/pindah, software editing-mu akan kehilangan jejak (Error "Media Offline").
