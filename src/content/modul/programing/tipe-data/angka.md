## Tipe Data Angka (Numeric)

Angka di dalam pemrograman biasanya dibagi menjadi dua jenis utama yang berlaku di hampir seluruh bahasa pemrograman (Python, Java, C++, dll):

### 1. Integer (Bilangan Bulat)
Sering disingkat `int`. Adalah tipe data angka yang tidak memiliki desimal/koma.
```python
umur = 20
jumlah_musuh = 5
```

### 2. Float (Bilangan Pecahan/Desimal)
Adalah tipe data angka yang memiliki titik desimal. (Perhatikan: dalam pemrograman, desimal dipisah menggunakan titik `.`, bukan koma `,`).
```python
berat_badan = 65.5
nilai_phi = 3.14
```

## Tipe Data Logika (Boolean)

Tipe data boolean adalah tipe data yang paling simpel tapi paling sering dipakai. Ia hanya bisa menampung 2 nilai: **Benar (True)** atau **Salah (False)**.
Nilai ini sangat krusial saat kita belajar struktur kontrol (If-Else) nanti.

### Dalam Python
Di Python, nilai Boolean menggunakan huruf kapital di awal.
```python
is_login = True
sudah_lulus = False
```

> [!TIP]
> **Komparasi Bahasa Lain:**
> Dalam JavaScript, C++, atau Java, tipe boolean ditulis dengan huruf kecil semua: `true` dan `false`.
